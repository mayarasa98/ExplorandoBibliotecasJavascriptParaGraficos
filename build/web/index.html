Exemplos de Bibliotecas de Gráficos


O objetivo desta atividade é familiarizar os alunos com diferentes bibliotecas JavaScript usadas para criar gráficos e ajudá-los a entender as características, vantagens e desvantagens de cada uma, de modo a implementar no nosso PBL. Exemplos: Chart.js, Highcharts, Plotly.js, dentre outros.
